# DSA Bootcamp - 108 Problems 🚀

## About
A structured DSA bootcamp covering 108 problems across core topics.
Solved in C++ with daily consistent practice.

## Topics Covered

### 📦 Arrays (1-23)
- Max sum row, Largest element, Frequency count
- Kadane's Algorithm, Trapping Rainwater, Spiral Matrix
- Dutch National Flag, Container With Most Water

### 🔤 Strings (24-41)
- Palindrome, Anagram, LCS
- Sliding Window, KMP, String Compression

### 🔗 Linked List (42-66)
- Cycle Detection, Merge Sorted Lists
- Flattening, Cloning, Knapsack

### 📚 Stack & Queue (67-91)
- Valid Parenthesis, Next Greater Element
- Stock Span, Sliding Window Maximum

### 🌳 Trees (92-103)
- Traversals, Diameter, LCA
- BST Validation, Zig-Zag, Serialization

### 🔍 Sorting & Searching (104-108)
- Bubble Sort, Binary Search
- Count Inversions, Linear Search
